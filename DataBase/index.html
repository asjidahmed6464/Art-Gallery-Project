What is this project about?

This is a basic ART Gallary Management System. Serious and successful art businesses run on a great amount of detailed information.

This project consists of a lot of interesting features including saving the details of various new artis, buyers, the artwork produced and even a mechanism of purchasing art.

Art Management

  • Hit the 'Register Artist' button if you're a new artist and you want to register yourself.
  • Hit the 'Register Buyer' button if you're a new buyer and you want to register yourself.
  • Hit the 'Register Art' button if you're a registered artist and you want to register an artwork you made.
  • Hit the 'Buy' button if you're a buyer and you want to buy an artwork.
